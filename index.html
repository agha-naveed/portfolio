  • OTP Generation: Generates a secure OTP for authentication.

  • Background Animation: A visually appealing moving gradient background.

Technologies Used

  • HTML: For the structure of the web page.
  • CSS: For styling the web page and creating the gradient animation.
  • JavaScript: For form validation, OTP generation, and handling the EmailJS API.
