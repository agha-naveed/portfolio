OTP Generation and Email Sending Application

This project is a web-based application designed to generate a One-Time Password (OTP) and send it to the user's email address. The interface collects the user's username and email, and upon submission, generates an OTP that is sent directly to the specified email if it is correct.

  • OTP Generation: Generates a secure OTP for authentication.
  • Email Sending: Sends the OTP to the user's email using the EmailJS.com API.
  • Form Validation: Ensures the email address is correctly formatted before sending the OTP.
  • Background Animation: A visually appealing moving gradient background.

Technologies Used

  • HTML: For the structure of the web page.
  • CSS: For styling the web page and creating the gradient animation.
  • JavaScript: For form validation, OTP generation, and handling the EmailJS API.
